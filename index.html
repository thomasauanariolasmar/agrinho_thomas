Do Campo a Cidade

O campo desde o princípio foi movido pelas mãos dos homens, por exemplo os povos da Mesopotâmia: os antigos sumérios, babilônios e assírios que foram os pioneiros no desenvolvimento de técnicas agrícolas, usando irrigação e o uso de arados. Mas é hora disso mudar, nessa nova sociedade tecnológica podemos facilitar o trabalho no campo usando máquinas como por exemplo o drone que é muito usado para mapear o campo muito mais rápido que um trabalhador humano, a tecnologia facilita muito o trabalho agrícola e é de importante ajuda.

O único problema que eu vejo é o que todo trabalhador manual sente medo, que é o medo de ser substituído por uma máquina e assim acabar perdendo seu emprego, mas mesmo a tecnologia facilitaria o trabalho deles, para não ficar para trás, o jeito é o trabalhador se reinventar no mercado de trabalho, estudando como usar as mãquinas, acredito que essa seja a melhor solução para melhorar as coisas no campo e ainda sim manter o emprego dos trabalhadores.
